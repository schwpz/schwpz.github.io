How Eyedra Works
1. Download & Launch Grab Eyedra.exe from Releases.
Double-click to start. No install needed.
2. Create Account (2FA Required) Register with username & strong password.
Scan QR with TOTP app & verify 2FA.
3. Get Wallet Code & Private Key Wallet code = public key (safe to share).
Private key is derived, never stored.
4. Login & Share Files Authenticate with 2FA. Split/encrypt files for others using their wallet code.
Only the intended recipient can decrypt.

What You Need To Remember
  • Your wallet code = your public key = your address for receiving encrypted files
  • Your private key = your access. Only you can decrypt files sent to your wallet code
  • Never share your credentials or private key. Share your wallet code to receive files
  • Files encrypted to a wallet code can only be opened by the private key owner
  • If you lose your credentials or wallet code, your files are lost forever
